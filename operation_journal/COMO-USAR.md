# Como usar o Operation Journal

Escreva de forma livre. Sem template obrigatório. Sem nome de agente.

---

## Para registrar uma experiência (sem criar conteúdo agora)

```
registrar: [o que aconteceu]
```

Exemplo:
```
registrar: hoje tentei usar o ChatGPT para criar a descrição de um produto.
Funcionou mas precisei revisar tudo. Levou uns 20min no total.
Ainda assim foi mais rápido do que escrever do zero.
```

---

## Para criar um post a partir de uma experiência

```
conteúdo: [o que aconteceu]
```

Exemplo:
```
conteúdo: hoje tentei usar o ChatGPT para criar a descrição de um produto.
Funcionou mas precisei revisar tudo. Levou uns 20min no total.
Ainda assim foi mais rápido do que escrever do zero.
```

---

## Guia rápido — o que dizer sem precisar lembrar nome de agente

| O que você quer | Diz assim |
|-----------------|-----------|
| Criar os posts da semana | "roda o ciclo" |
| Criar um post específico | "cria um post sobre X" |
| Registrar uma experiência | "registrar: [o que aconteceu]" |
| Transformar experiência em post | "conteúdo: [o que aconteceu]" |
| Ver o calendário da semana | "mostra o calendário" |
| Ver os roteiros gerados | "mostra os roteiros" |
| Ver o que está na fila | "mostra a fila" |
| Salvar uma ideia para depois | "ideia: [a ideia]" |

---

## Sobre os arquivos desta pasta

Cada entrada salva aqui fica disponível para virar conteúdo no futuro.
Quando você rodar um novo ciclo, o sistema lê os registros e usa como base.

Não precisa apagar nada — só acumula.

---

## Formato dos arquivos

Os arquivos são salvos automaticamente como:

`AAAA-MM-DD-[resumo].md`

Você não precisa criar os arquivos manualmente.
Só diga "registrar:" no chat e o sistema cuida do resto.
