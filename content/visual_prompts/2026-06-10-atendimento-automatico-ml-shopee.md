**Objetivo Visual:** Entregar um ativo reutilizável — templates que o seller salva, copia e usa hoje.

**Emoção Principal:** Simplicidade — "posso usar isso agora, sem esforço"

**Hierarquia Visual:**
1. O prompt completo para gerar templates com IA
2. Exemplo de template pronto para a pergunta mais comum
3. Instrução de adaptação
4. CTA para salvar

**Âncora Visual:** Interface de IA (ChatGPT/Claude com prompt preenchido)

## Blueprint Visual

### Formato
Carrossel complementar — 6 slides (1080x1080px)

### Slide 1 — Capa
- Headline: "O prompt que uso para criar templates de atendimento com IA"
- Sub: "copie, adapte, cole no ML e Shopee"
- Fundo: #1E293B
- Título: Montserrat ExtraBold, #FFFFFF
- Destaque: "prompt" em #F97316
- Elemento: mockup de interface de IA com texto em campo de input

### Slide 2 — O Prompt Base
- Título: "O prompt completo"
- Conteúdo: o prompt formatado em bloco de código visual
  ```
  Você é um assistente de atendimento de e-commerce brasileiro.
  Gere um template de resposta automática para a pergunta:
  "[COLE A PERGUNTA FREQUENTE AQUI]"
  
  Regras:
  - Tom cordial, direto, sem exageros
  - Máximo 3 linhas
  - Inclua próximo passo (ex: "acesse o anúncio para...")
  - Não use emojis em excesso
  ```
- Fundo: #1E293B (bloco de código)
- Texto: #F8FAFC (monoespaçada)
- Fundo do slide: #F8FAFC

### Slide 3 — Template Exemplo 1
- Título: "Pergunta: 'O produto serve para [medida X]?'"
- Template gerado:
  > "Olá! Sim, o produto atende essa medida. As dimensões completas estão na descrição do anúncio. Qualquer dúvida, estamos à disposição!"
- Fundo: #F8FAFC
- Bloco do template: borda esquerda #2563EB, fundo #EFF6FF

### Slide 4 — Template Exemplo 2
- Título: "Pergunta: 'Qual o prazo de entrega?'"
- Template gerado:
  > "Olá! O prazo aparece automaticamente no anúncio conforme seu CEP. Em média, entregas para capitais chegam em 3–5 dias úteis. Dúvidas, é só falar!"
- Fundo: #F8FAFC
- Bloco do template: borda esquerda #2563EB, fundo #EFF6FF

### Slide 5 — Como Adaptar
- Título: "Como adaptar para o seu produto"
- 3 regras simples:
  1. Troque o genérico pelo específico (ex: "produto" → "organizador de gaveta")
  2. Inclua o número real de dias de entrega da sua operação
  3. Teste o template com 3 compradores antes de automatizar
- Fundo: #F8FAFC
- Numeração: #F97316

### Slide 6 — CTA
- "Salva esse carrossel + comenta 'ATENDIMENTO' para receber os outros 5 templates"
- Fundo: #2563EB
- Texto: #FFFFFF, Montserrat Bold
