## Resumo

Tema: O que automatizar numa operação solo

Pilar: Educação

Formato Principal: Carrossel

Mensagem Principal: Alta frequência + baixa criatividade = automatize primeiro.

---

## Estratégia Escolhida

**EXPANSÃO**

O carrossel entrega o framework. O estático complementar aplica o framework na prática — uma tabela visual com as tarefas mais comuns de operação solo classificadas pelo critério ensinado.

---

## O Que Já Foi Explicado

- O critério de decisão: frequência × criatividade
- Lista de tarefas automatizáveis
- Lista de tarefas que não dão para automatizar
- Por onde começar

## O Que Será Entregue Agora

- Tabela visual aplicando o critério a 10 tarefas comuns de marketplace
- Classificação visual imediata: automatize / manual / depende

---

## Blueprint Visual

### Formato
Post estático — 1080x1080px

### Layout
- Título: "10 tarefas de marketplace — automatiza ou não?"
- Tabela com 3 colunas: Tarefa | Frequência | Criatividade → Decisão
- Cada linha com código de cor na decisão: verde (#2563EB) = automatize, vermelho (#EF4444) = manual, amarelo (#F97316) = depende

### Tarefas
| Tarefa | Decisão |
|--------|---------|
| Criar título de anúncio | ⚡ Automatize |
| Escrever descrição | ⚡ Automatize |
| Gerar imagem de produto | ⚡ Automatize |
| Responder pergunta repetitiva | ⚡ Automatize |
| Controle de estoque | ⚡ Automatize |
| Negociação com fornecedor | 🧠 Manual |
| Decisão de comprar produto novo | 🧠 Manual |
| Responder reclamação | 🧠 Manual |
| Definir preço estratégico | 🧠 Manual |
| Foto do produto | ⚠️ Depende |

### Cores
- Fundo: #F8FAFC | Barra título: #1E293B
- ⚡ Automatize: #2563EB | 🧠 Manual: #EF4444 | ⚠️ Depende: #F97316
