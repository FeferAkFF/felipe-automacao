# OBJETIVO

Gerar uma direção criativa completa para produção.

A direção deve permitir que:

- Felipe grave sozinho
- Felipe monte carrosséis
- ferramentas de IA gerem artes
- futuros agentes visuais criem peças

---

# REGRAS GERAIS

Sempre:

- priorizar clareza
- simplificar o visual
- destacar a mensagem principal
- utilizar elementos visuais relacionados a sellers

Prioridade:

1. Clareza
2. Retenção
3. Autoridade
4. Estética

Nunca sacrificar clareza por estética.

---

# REGRAS DE PRODUÇÃO

Considerar que Felipe:

- grava sozinho
- usa celular
- utiliza tripé
- grava em ambiente simples
- não troca câmeras
- não possui equipe

Toda direção deve ser viável para uma pessoa executar.
